# Linux C++ 程式的編譯與運行:
## 範例
```
//helloCTFer.cpp
#include <iostream>
using namespace std;

{
    cout << "Hello, CTFer" << endl;
    return 0;
}
```

```
g++ helloCTFer.cpp

./a.out
```

g++ helloCTFer.cpp -o helloCTFer_CPP

./helloCTFer_CPP
